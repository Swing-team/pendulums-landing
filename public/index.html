FEATURES

Learn More About Pendulums

OPEN SOURCE

Pendulums would be always free to use without any limits. The source code for web, desktop and mobile apps are available at Github.
If you like Pendulums, please give it a star ;)

OFFLINE MODE

If you have a weak internet connection, don't worry you can still track your time when you are offline and the data would be synced with server when the connection is available.

SIMPLE AND EASY

We tried to keep everything as simple as possible. Pendulums is suitable for daily activities as well as freelancing tasks. Just create a project and start an activity on it.

WORK WITH TEAMMATES

You can share your project with your teammates in different roles and track your team members as an owner or admin.

SHOW SOME LOVE

We truly thank you for choosing us. It is an honor for us to have lovely users like you. Thank you for reporting issues, feature requests, your feedbacks and helping us to improve Pendulums. As you know every product needs to be funded to be alive during its journey, We have a lot of nice features which are planned to be added to Pendulums as soon as possible and absolutely maintaining Pendulums. We are asking you to show us your love if you really like Pendulums and want to see its progress.

OUR STORY

We started our work in Feb. 2017. After a few weeks of working on our other projects, we found that we need to manage our time to get the best from it.

We tried other time tracking tools out there but none of them fit into our requirements. We needed a "free" tool which allows an unlimited number of projects and users per project and has a desktop app in all platforms with a simple interface that our non-technical team members can work with easily.

So we decided to create another tool ourselves and the Pendulums was born.
