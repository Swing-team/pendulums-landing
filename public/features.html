Time Flies

Manage and track your time more efficiently using a modern time tracking tool.

Pendulums is a free time tracking tool which helps you to manage your time in a better manner with an easy to use interface and useful statistics.

The source code for web, desktop and mobile apps are available at Github
If you like Pendulums, please give it a star ;)

Unlimited Projects

You can create as many projects as you want. Here we don't have any limitations to create projects.

User Friendly Charts And Reports

You can see a user friendly chart in activities page; Using this chart, you can see how much time your team members have spent on the project and analyze the performance of your team members and the progress of your project. You can also filter the results by time and members who are involved in your project. You can export activities in a CSV file or backup entire project activities in a json file.

Unlimited Team Members

Feel free to invite as many members as you want to your projects. At anytime you can grant admin permission to them or remove them from your projects.

Unlimited Notes

We thought it's a good idea to keep track of ideas, todos, code snippets, etc. in the same app that you track your time for your projects. So Pendulums allows you to take as many notes as you want and tag them for a specific project.

We All Want To Live Healthy

If you don't rest enough, You won't be healthy, which results to not being productive in your work. In Pendulums you can set a rest time reminder in your profile settings. Pendulums will notify you to rest in the specified time intervals. You can choose your preference between standard presets or define your own customized notification intervals.
